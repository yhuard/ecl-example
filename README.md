# ECL example

Requirements:
-   Node.js >= 6.9.5
-   yarn >= 0.20.3

## Setup

```
yarn
```

## Build for development

```
yarn start
```

If you don't want to use the integrated change watcher, you can also just run:

```
yarn build
```

## Build for production use

```
yarn dist
```

## Lint

```bash
yarn lint
```
